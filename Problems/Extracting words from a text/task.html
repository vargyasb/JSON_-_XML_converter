Extracting words from a text Open on Hyperskill

You need to write a program to extract all words that includes the "program" substring from a given text. The text can be large enough. You should not take whitespaces, punctuation mark and other special characters as a part of the extracted word.

Try to use Matcher and Pattern to solve it. All needed import is already here.

Input data format

A single string representing a text about programs, programmers or programming.

Output data format

Multiple lines where each line contains two parts separated by one space - the start index of a word and then, this word.


Sample Input:
All Java programmers program good programs.
Sample Output:
9 programmers
21 program
34 programs


Memory limit: 256 Mb
Time limit: 5s
